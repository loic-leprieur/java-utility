# loic-leprieur's java-utilty

## What's the purpose of this repo ?
Record useful utility classes and samples of code for reuse in any Java project

## PMD

This Maven project use the PMD tool for checking Java coding style errors added inside the codebase.
It is executed at each `maven clean install`
https://pmd.github.io/

## Lombok

Lombok is required so you have to install the plugin for your IDE (works with Eclipse and IntelliJ) : https://projectlombok.org/

## Can I reuse or copy code from this repository ?

Of course you can! I mainly use it for keeping a trace of my breakthrough but anyone can fork, copy or extend this code (please send a pull request if you have a better approach than
mine, I would be grateful!)
